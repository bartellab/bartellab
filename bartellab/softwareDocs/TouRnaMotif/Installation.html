Download/Installation (back to Introduction)


This module can be used by any Python interpreter. It is compatible with Python 2.3.4 and will likely work with most versions of Python; however, because the Python programming language is not developed with a strong "do-no-harm" philosophy, it is possible that previous or future versions of Python will be incompatible with the 21U-RNA motif scoring module as implemented here.

The source code files can be downloaded below. These files should be placed into the working directory from which they will be used. The 21U-RNA motif scoring module can then be loaded by typing "import touScoreModule" into your script or interactive python session, and the motif scoring matrix module can be loaded independently by typing "import motifModule".

Download by right-clicking on the links below and selecting "Save as". Use the linux applications gzip and tar to decompress the file packages.


  • Documentation contains all of the doc pages found here, along with all of the associated file packages. Download, unzip into a host directory, and use this documentation locally. Start by opening the file "Introduction.html" in your web browser.
  • All scripts contains all of the python files necessary for using the touScoreModule. It includes:
    • touScoreModule.py contains the 21U-RNA motif scoring matrixes and functions for applying them.
    • motifModule.py contains methods for constructing motif matrix representations.
    • It also includes the test sequence file: testSequences.py
