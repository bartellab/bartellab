Downloads (back to Introduction)

Download by right-clicking on the links below and selecting "Save as".

Use the linux applications gzip and tar to decompress the file packages.


  • Documentation contains all of the doc pages found here, along with all of the associated file packages. Download, unzip into a host directory, and use this documentation locally. Start by opening the file "Introduction.html" in your web browser.
  • All scripts contains all of the python scripts necessary for running mirscan. It includes:
    • mirscanTrainer.py
    • mirscanExecute.py
    • mirscanModule.py
    • score_cut.py
    • oneSeq.prototype.py
    • twoSeq.prototype.py
  • Sample files contains sample files for each of the file formats used by mirscan. These files can be used to conduct tests following installation. It includes:
    • oneSeq.round0.train: one organism training/foreground.
    • twoSeq.round0.train: two organism training/foreground.
    • sample1.round0.fax: one organism background.
    • sample2.round0.fax: two organism background.
    • oneSeq.round0.matrix: one organism scoring matrix made using oneSeq.rubyEtAl.py.
    • twoSeq.round0.matrix: two organism scoring matrix made using twoSeq.rubyEtAl.py.
    • oneSeq.round0.scr: one organism foreground scoring results.
    • twoSeq.round0.scr: two organism foreground scoring results.
    • sample1.round0.scr: one organism background scoring results.
    • sample2.round0.scr: two organism background scoring results.
    • sample1.round1.fax: one organism background after one round of cutting.
    • sample2.round1.fax: two organism background after one round of cutting.
    • The scoring criteria used in Ruby et al., Genome Res. 2007.
      • oneSeq.rubyEtAl.py: single organism criteria.
      • twoSeq.rubyEtAl.py: two ortholog criteria.
  • Scoring matrices from Ruby et al., Genome Res., 2007.
